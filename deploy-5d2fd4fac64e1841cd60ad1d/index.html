About Me

Here are a few of my projects.

Airbnb Press Clone

This is one of the first projects that I completed for my bootcamp. I had to choose pictures for the site online, and decided to use unsplash.com. I chose the photos that I felt best represented the related content, and then coded the site using HTML and CSS.

Currency Exchange App

This is one of my favorite projects, and one of the first times I had ever used React. This app uses a free API to get its data ( Alt Exchange Rates ). The app is hosted for free on my Heroku account, and the API goes to sleep if it has not been used for a while, so please be patient. This app uses Bootstrap, React, FontAwesome, React Router, and a photo from unsplash. I thoroughly enjoyed working on the project, and look forward to using React in the future.

Airbnb Clone

This is a fun project that I completed for my bootcamp. As opposed to the first Airbnb press clone, this project is a clone of the Airbnb booking site. This project focused on the front end only, and was built using HTML and CSS.

10 Second Math Game

This is a pretty simple project that I built using jQuery. I built this for an assignment in my bootcamp. After I met all of the requirements, I decided to add a few extra features like the ability to choose the maximum number used, and the changing stop-light colors for the time. Overall, I am pretty happy with the project.

Movie Finder App

This was a very useful and enjoyable app to build. This was one of the first times I used AJAX to build an app. This app also uses React for the front end. The API used in this project is The Open Movie Database. The search results are also linked to IMDB, and clicking on a movie title will open a new tab with full movie details.
